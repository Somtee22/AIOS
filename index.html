What does All-In-One Security do?

Included in Free and Premium:

Login Security

A password strength tool, two-factor authentication and the ability to hide your login page from bots protects your site from brute force attacks. Lockout IP addresses, block malicious users and protect customer data with a comprehensive suite of free login security features.

Firewall & File Protection

We afford the same firewall and file protections to both our free and premium customers. AIOS firewall adds rules to your .htaccess file to deny access to both itself and your wp-config.php file. Disable PHP file editing, lockout IP addresses, blacklist user agents and more.
